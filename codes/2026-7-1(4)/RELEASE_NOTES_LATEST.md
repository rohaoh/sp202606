# 🪂 낙하 물리 시뮬레이터 — 릴리즈 노트

**버전:** `3.0.0`  ·  **플랫폼:** Windows (x64, exe/msi)  ·  **빌드:** Electron + C++(N-API) 물리 엔진

하늘에서 무언가 떨어진다면? **종단속도 · 충격량 · 파괴율**을 실시간 3D로 계산·시각화하는 시뮬레이터입니다.

---

## ✨ v2.0.0 → v3.0.0 핵심 변경 사항

### 1. 물리 엔진 고도화
- **Reynolds 수 기반 드래그**: 저속(Re<1000) 구간에서 Cd를 동적으로 보정해 정확도 향상
- **물 저항**: 수심(waterDepth) 입력 시 공기보다 20배 강한 저항력 적용
- **Coriolis 효과**: 위도 입력에 따른 지구 자전 편향 계산 (동/서 편향 거리로 표시)
- **고도별 중력 변화**: WGS84 모델 기반, 극지방/적도 중력 차이와 고도에 따른 감소 반영
- **회전 감쇠**: 공기 저항으로 스핀(RPM)이 시간에 따라 점차 줄어듦
- **바운스 물리**: 재충돌 시 에너지 손실 계산, 연쇄 바운스 횟수/속도 추적
- **에너지 손실 추적**: 낙하 전체 과정의 기계에너지 손실량 누적 기록

### 2. 다중 물체 시간차 낙하 + 물체간 충돌 (신규)
- Object 2/3/4 각각에 **낙하 시간차(초)** 입력 — 동시 낙하가 아니라 순차적으로 떨어뜨리는 연출 가능(기본값 0 = 기존과 동일)
- **시간차 자동 적용** 버튼으로 0, step, 2×step 초 간격을 일괄 설정
- **물체간 충돌(시각 효과)** 토글(기본 OFF) — 물체끼리 겹치면 서로 밀어내는 시각적 보정. 핵심 물리 계산·충돌 결과 수치는 그대로 유지됨
- 다중 물체 낙하 계산을 **Web Worker 풀로 병렬 처리** — 물체 수가 늘어나도 메인 스레드가 버벅이지 않음

### 3. 프리셋 · 결과 내보내기 강화
- 프리셋을 **JSON 파일로 내보내기/가져오기** (기존 localStorage 저장/불러오기에 추가) — 다른 PC와 공유하거나 백업 가능
- CSV 내보내기에 **다중 물체 요약 섹션** 추가: 물체별 질량·면적·Cd·시간차·종단속도·충돌속도·낙하시간·착지시각

### 4. 전체적인 UI 다듬기
- 다크 테마에 맞춘 통일된 스크롤바
- 키보드 포커스 표시(`:focus-visible`) — 키보드 탐색 접근성 개선
- 버튼 클릭 시 눌림 피드백 애니메이션
- Multi-Object/히스토리 카드 hover 시 테두리·그림자 강조
- 좌측 패널 섹션 접힘/펼침 부드러운 전환

### 5. 자동 업데이트
- Help 메뉴에 **Check for Updates…**(수동 확인) + **시작 시 자동으로 업데이트 확인**(토글, 기본 OFF) 추가
- GitHub Releases를 배포처로 사용 — **이 v3.0.0 릴리즈부터 실제로 동작**
- exe(NSIS)로 설치한 경우에만 자동 업데이트 지원. msi는 자동 업데이트 대상이 아니며, 새 버전이 나오면 GitHub Releases에서 새 msi를 받아 수동 재설치 필요

### 6. 배포 자동화 (개발 인프라)
- **GitHub Actions로 태그 push / 수동 실행 기반 자동 빌드+배포** 구축 — Windows 클라우드 러너에서 `npm install` → C++ 컴파일 → 번들링 → exe/msi 패키징 → GitHub Releases 업로드까지 전 과정이 자동으로 처리됨
- **exe(NSIS) + msi 두 가지 설치 형식 동시 제공** — 개인 배포(exe, 자동 업데이트 지원)와 사내/그룹정책 배포(msi) 모두 지원
- **앱 아이콘 신규 추가** (다크 테마에 맞춘 낙하공 모티프) — 이전에는 아이콘이 없어 Electron 기본 아이콘으로 표시되고 MSI 빌드도 실패했음

### 7. 코드 품질 · 안정성 (개발자용)
- TypeScript 타입 체크 활성화(`checkJs`) — 검증 과정에서 실제 잠재 버그 2건 발견 및 수정
  (죽어있던 중복 코드 두 곳: `main.js`의 불필요한 재-require, `parallel-physics.js`의 중복 선언되어 항상 덮어써지던 버그 함수)
- C++ 물리 엔진 핵심 계산에 대한 **단위 테스트 24개** 추가 (cmake-js/Electron 헤더 없이 `g++`만으로 독립 실행 가능)
- 빌드 캐싱(변경 없으면 C++ 재컴파일 스킵), 보안 감사, 번들 분석, 성능 벤치마크, 메모리 누수 감지, API 문서 자동 생성 등 개발 도구 10종 추가
- CI 환경에서 `npm install`이 항상 실패하던 `husky` 관련 버그 수정

## 🧩 전체 기능

> 모든 기능은 좌측 패널/툴바의 **토글 스위치**로 개별로 켜고 끌 수 있습니다.

### 물리 시뮬레이션
- **종단속도 / 운동방정식**: `v_t = √(2mg / ρCdA)`, `ma = mg − ½ρCdAv²`
- **ISA 표준 대기**: 고도별 공기 밀도·온도·압력 자동 계산
- **바람 / Magnus 효과 / 발사체 모드 / 지형(평지·경사면·수면·고지대)**
- **공력 가열**: 정체 온도 · Sutton-Graves 열유속 · 복사 평형 표면 온도
- **다중 물체**: 최대 3개 물체 독립 시뮬레이션 + **시간차 낙하** + **물체간 충돌(시각 효과)** (v3.0.0 신규)
- **Reynolds 수 기반 드래그 보정 · 물 저항 · Coriolis 효과 · 고도별 중력 · 회전 감쇠 · 바운스 에너지 손실** (v3.0.0 신규)

### 충돌 · 파괴
- **임계 기반 파괴**: 충격 압력이 재질 항복강도를 넘을 때까지 버팀(Withstood), 넘으면 점진적 파괴
- **타깃 두께·항복강도 직접 입력**, **버팀 시 바운스**, **충격 에너지 대시보드**
- **파편 / 분화구 / 먼지**: InstancedMesh 기반 고성능 렌더링, 파편끼리 충돌 판정
- **재료별 파편 차등**(유리=잘게, 금속=3조각 분리, 목재/콘크리트=청크), **타겟 바닥에서 띄우기**

### 3D 모델 (GLB / STL)
- 프리셋 모델 3종, 파일 업로드(assets 자동 복사), 모델 크기 조정(비율 유지), Multi-Object GLB

### 시각화 · 편의
- 카메라 프리셋/추적, 실시간 그래프, 단위 전환(SI/km·h/제국단위), 궤적 라인·자료 표·CSV/PNG 내보내기
- **프리셋 저장/불러오기(localStorage) + JSON 파일 내보내기/가져오기**(v3.0.0 신규)
- 온디맨드 녹화/재생, 설정창 폭 드래그 조절
- 다크 테마 통일 스크롤바·포커스 표시·클릭 피드백·hover 강조(v3.0.0 신규)

### 배포 · 업데이트 (v3.0.0 신규)
- **자동 업데이트**(electron-updater, exe 전용) · **GitHub Actions 자동 배포**(태그/수동 트리거) · **exe + msi 동시 제공**

## 💾 설치 및 실행 (Windows · x64)

> 처음 한 번만 Node.js, CMake, Visual Studio Build Tools(C++) 설치가 필요합니다.

```bash
cd codes/2026-7-1(4)
npm install
npm run build-addon   # C++ 물리 엔진 컴파일 (cmake-js)
npm start
```

설치 파일(exe + msi)로 배포하려면:

```bash
npm run dist           # dist/ 폴더에 exe/msi 생성 (로컬, 업로드 없음)
npm run release        # 위와 동일 + GitHub Releases에 자동 업로드 (GH_TOKEN 필요)
```

또는 저장소의 **Actions 탭 → Release 워크플로우 → Run workflow**로 클라우드에서 직접 빌드/배포할 수 있습니다.

> ⚠️ C++ 물리 코드를 수정한 경우 반드시 **`npm run build-addon` 재실행** 후 실행하세요.
