# 🚀 깃허브(GitHub) 협업 시작하기 가이드

우리 협업을 위해 딱 필요한 핵심만 정리했어! 깃허브가 처음이어도 이 가이드만 따라오면 문제없어.

## 1. 깃허브의 기본 개념
* **main 브랜치:** 최종 결과물이 저장되는 메인 공간 (직접 수정 금지!)
* **개인 브랜치:** 각자 작업하는 개인 작업실

## 2. HTML: 웹페이지의 뼈대 (기초 태그)
HTML은 태그(`< >`)를 사용해서 웹페이지의 구조를 잡아.
* `<h1>제목</h1>`: 가장 큰 제목
* `<p>본문 내용</p>`: 문단 작성
* `<a href="주소">클릭 시 이동</a>`: 하이퍼링크 만들기
* `<img src="이미지주소">`: 이미지 추가

## 3. Markdown(md): 문서 작성법
깃허브에서 기록을 남길 때 사용하는 쉬운 문법이야.

| 기능 | 문법 | 결과 |
| :--- | :--- | :--- |
| 제목 | `# 제목` | 가장 큰 제목 |
| 강조 | `**글씨**` | **굵게** |
| 리스트 | `- 내용` | • 내용 |
| 링크 | `[이름](주소)` | [클릭 가능한 링크] |
